# CCPA (ccpa)
The California Consumer Privacy Act (CCPA) is a state statute intended to enhance privacy rights and consumer protection for residents of California. It provides consumers with rights to know what personal information is collected, delete personal data, opt-out of data sales, and non-discrimination for exercising privacy rights.
